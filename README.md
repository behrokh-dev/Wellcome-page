
# 👋 Welcome Page

A simple and interactive **Welcome Page** built with **HTML, CSS, and JavaScript**.

This project was created as one of my first JavaScript exercises to practice working with user input, DOM manipulation, and time-based logic. The application greets users by their name and displays an appropriate message based on the current time of day.

## 🚀 Live Demo

🔗 https://behrokh-dev.github.io/Wellcome-page/

## ✨ Features

- 👤 Personalized greeting using the user's name
- 🌅 Displays "Good Morning" in the morning
- ☀️ Displays "Good Afternoon" during the day
- 🌇 Displays "Good Evening" in the evening
- 🌙 Displays "Good Night" at night
- ⚡ Dynamic content updates
- 📱 Responsive and user-friendly design

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

## 🧠 What I Learned

While building this project, I practiced:

- Handling user input
- DOM manipulation
- Working with JavaScript Date objects
- Using conditional statements
- Dynamically updating page content

## 🎯 Purpose

This project was created for learning purposes and represents one of my first steps into JavaScript development. It helped me understand how to create interactive pages and implement time-based greetings.
